Release Notes

Version 1.0.1

1. Manual operations needed before initial sync sharded cluster

This applies to MongoDB - MongoDB replication.

In order to improve the initial sync speed, Tapdata connects to each shard directly to perform initial sync. This may causes orphaned documents to be replicated to the target cluster. To prevent orphaned documents to be copied, you may perform the following tasks before starting the replication job: 

    - Stop balancer. Refer to https://docs.mongodb.com/manual/reference/method/sh.stopBalancer/
    - Clean orphaned documents. Refer to https://docs.mongodb.com/manual/reference/command/cleanupOrphaned/

2. Initial sync cannot be resumed upon interruption when not using custom SQL

While Tapdata in general can resume the replication after the job was stopped or interrupted, it cannot resume an initial sync if custom SQL is not used. If the job was stopped during the initial sync phase for some reason, upon its restart, Tapdata will start from beginning to read the data from source database.

3. Initial sync cannot be resumed when not using custom SQL

When running the replication job from relational database to MongoDB, if the job was interrupted during the initial load, Tapdata will start from the beginning when the job is restarted.

The exception to this is if you are using Custom SQL for loading the data from relational database. In which case the job may be able to resume from the point it was interrupted.

4. Pause job before performing DDL operations on source database.

Oracle/MySQL/SQLServer DDL operations are not yet supported during CDC replication. Before running the DDL operation(such as drop table, alter table etc), be sure to stop the replication job first. You may resume the job after the DDL operation is completed.

5. DEV-1013: Replication progress may not work properly when there are tables without primary key

Tables without primary keys are not replicated under datatabase clone mode. When this happens, the replication progress may for ever show a lag between source and target.
